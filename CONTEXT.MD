# GitHubPH — Project Context

## 🎯 Mission

GitHubPH is a **free, gamified, interactive learning platform** designed specifically for **Filipino students, beginners, capstone groups, and aspiring developers** who want to learn Git and GitHub.

The goal is not to create another GitHub documentation website.  
The goal is to create **the easiest and most enjoyable place for Filipinos to finally understand Git and GitHub.**

---

## 📖 Product Vision

**Think:** Duolingo + GitHub + Interactive Simulations + Filipino Learning Experience

**Not:** Traditional documentation website.

---

## 🎯 Target Users

### Primary

- BSIT Students
- Capstone Groups
- Beginners Learning GitHub

### Secondary

- Self-taught Developers
- Web Developers
- Unity Developers
- Career Shifters

---

## 🏛️ Product Pillars

| Pillar        | Description                                             |
| ------------- | ------------------------------------------------------- |
| **Learn**     | Structured lessons with quizzes and Filipino context    |
| **Practice**  | Interactive Git simulations (playground, branch, merge) |
| **Progress**  | XP, levels, achievements, streaks, leaderboards         |
| **Apply**     | Real-world workflows and capstone project scenarios     |
| **Community** | Future social learning features (forums, study groups)  |

## 🧩 Major Systems

### Learning System

Courses → Lessons → Learning Paths → Roadmaps → Quizzes

### Gamification System

XP → Levels → Streaks → Achievements → Missions → Leaderboards

### Simulation Engine

Git Playground (terminal) + Branch Simulator + Merge Conflict Simulator + PR Simulator

### User Management

Auth (GitHub + Google OAuth) → Profiles → Progress Tracking

### Analytics

User Engagement → Lesson Completion → Retention Metrics → Event Tracking

### AI Layer (Future)

GitBuddy AI Tutor → Repository Analyzer → Personalized Learning → Code Review Assistant

## ✨ WOW Factors

1. **Interactive Git Playground** — Live terminal in browser, learn visually
2. **Branch Simulator** — Visual branch creation and merging
3. **Merge Conflict Simulator** — Interactive conflict resolution
4. **Mission-Based Learning** — Lessons become quests
5. **XP & Achievement System** — Gamification that motivates
6. **Filipino Context Learning** — Capstone projects, library systems, POS, attendance systems
7. **Contribution Heatmap** — GitHub-style activity visualization
8. **Level-Up Celebrations** — Confetti and animations

## 🎨 Brand Identity

### Color Palette (Filipino-Inspired)

- **Primary Blue** (#2563EB) — Trust, technology, reliability
- **Accent Yellow** (#F59E0B) — Energy, optimism, Filipino sun
- **Passion Red** (#DC2626) — Passion, excitement
- **Growth Green** (#16A34A) — Progress, learning, success

### Typography

- **Inter** — Body text, headings (clean, modern, readable)
- **JetBrains Mono** — Code blocks, terminal, Git commands

### Voice & Tone

- **Friendly & Encouraging** — "Kaya mo 'yan!" energy
- **Filipino-English (Taglish)** — Accessible, relatable explanations
- **Beginner-First** — No assumptions of prior knowledge
- **Fun but Professional** — Gamified but skill-focused
